DotCloud for Windows

Simple installer to setup DotCloud CLI (Command-Line Interface) on Windows.
The setup includes all prerequisites (Python, Cygwin, etc.) to run DotCloud.

Download binaries

For more information read the official docs.

To build from source you need:

  • Python 2.7 (x86)
  • py2exe 0.6.9 (x86)

You can compile the installer with Inno Setup.
